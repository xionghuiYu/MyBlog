# Tailwind CSS 快速入门

Tailwind CSS 是一个实用的 CSS 框架，让你可以快速构建现代网站。

## 安装

```bash
npm install -D tailwindcss postcss autoprefixer
npx tailwindcss init -p
```

## 基本用法

```html
<div class="bg-blue-500 text-white p-4 rounded-lg">
  Hello Tailwind!
</div>
```

## 响应式设计

Tailwind 提供了强大的响应式设计支持。

## 总结

Tailwind CSS 让你不必离开 HTML 就能构建完全定制的网站设计。
