# 项目基础与设计系统实现计划

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 创建项目基础架构，配置 Vite + Vue 3 + Tailwind CSS，建立设计系统（色彩、字体、间距、圆角）。

**Architecture:** 使用 Vite 作为构建工具，Vue 3 Composition API 作为前端框架，Tailwind CSS 作为样式框架，通过 CSS 变量实现主题系统，Google Fonts 提供字体。

**Tech Stack:** Vite, Vue 3, Tailwind CSS, Google Fonts (Archivo + Space Grotesk)

### Task 6: 创建主题状态管理

**Files:**
- Create: `src/stores/theme.js`

- [ ] **Step 1: 创建 theme.js**

```javascript
import { ref, watch } from 'vue'

const STORAGE_KEY = 'blog-theme'

const isDark = ref(false)

const initTheme = () => {
  const saved = localStorage.getItem(STORAGE_KEY)
  if (saved) {
    isDark.value = saved === 'dark'
  } else {
    isDark.value = window.matchMedia('(prefers-color-scheme: dark)').matches
  }
  applyTheme()
}

const applyTheme = () => {
  if (isDark.value) {
    document.documentElement.classList.add('dark')
  } else {
    document.documentElement.classList.remove('dark')
  }
}

const toggleTheme = () => {
  isDark.value = !isDark.value
  localStorage.setItem(STORAGE_KEY, isDark.value ? 'dark' : 'light')
  applyTheme()
}

watch(isDark, (newValue) => {
  localStorage.setItem(STORAGE_KEY, newValue ? 'dark' : 'light')
  applyTheme()
})

initTheme()

export const useTheme = () => ({
  isDark,
  toggleTheme
})
```
